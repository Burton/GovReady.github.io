--- layout: control permalink: /CCE-27124-7/ name: CCE-27124-7 title: Disable SSH Support for .rhosts Files ident: CCE-27124-7 severity: medium ---

SSH trust relationships mean a compromise on one host can allow an attacker to move trivially to other hosts.

Failing to disable ssh support for .rhosts files is of medium severity.

View all 11 Configure OpenSSH Server if Necessary rules.

Technical Description

SSH can emulate the behavior of the obsolete rsh command in allowing users to enable insecure access to their accounts via .rhosts files.

To ensure this behavior is disabled, add or correct the following line in /etc/ssh/sshd_config: 

IgnoreRhosts yes
