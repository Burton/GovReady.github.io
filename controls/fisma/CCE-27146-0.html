Failing to disable squid is of low severity.

Rationale

Running proxy server software provides a network-based avenue of attack, and should be removed if not needed.

via FISMA SSG. View all 2 Disable Squid if Possible rules.

Technical Description

The squid service can be disabled with the following command: # chkconfig squid off
