Failing to disable portreserve (portreserve) is of low severity.

Rationale

The portreserve service provides helpful functionality by preventing conflicting usage of ports in the reserved port range, but it can be disabled if not needed.

via FISMA SSG. View all 24 Base Services rules.

Technical Description

The portreserve service is a TCP port reservation utility that can be used to prevent portmap from binding to well known TCP ports that are required for other services. The portreserve service can be disabled with the following command: # chkconfig portreserve off
