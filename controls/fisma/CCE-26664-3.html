--- layout: control permalink: /CCE-26664-3/ name: CCE-26664-3 title: Record Events that Modify User/Group Information ident: CCE-26664-3 severity: low ---

In addition to auditing new user and group accounts, these watches will alert the system administrator(s) to any modifications. Any unexpected users, groups, or modifications should be investigated for legitimacy.

Failing to record events that modify user/group information is of low severity.

View all 14 Configure auditd Rules for Comprehensive Auditing rules.

Technical Description

Add the following to /etc/audit/audit.rules, in order to capture events that modify account changes: 

# audit_account_changes
-w /etc/group -p wa -k audit_account_changes
-w /etc/passwd -p wa -k audit_account_changes
-w /etc/gshadow -p wa -k audit_account_changes
-w /etc/shadow -p wa -k audit_account_changes
-w /etc/security/opasswd -p wa -k audit_account_changes
