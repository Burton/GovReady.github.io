--- layout: control permalink: /CCE-26476-2/ name: CCE-26476-2 title: Verify All Account Password Hashes are Shadowed ident: CCE-26476-2 severity: medium ---

The hashes for all user account passwords should be stored in the file /etc/shadow and never in /etc/passwd, which is readable by all users.

Failing to verify all account password hashes are shadowed is of medium severity.

View all 4 Verify Proper Storage and Existence of Password Hashes rules.

Technical Description

If any password hashes are stored in /etc/passwd (in the second field, instead of an x), the cause of this misconfiguration should be investigated. The account should have its password reset and the hash should be properly stored, or the account should be deleted entirely.
