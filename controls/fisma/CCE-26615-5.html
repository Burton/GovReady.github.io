--- layout: control permalink: /CCE-26615-5/ name: CCE-26615-5 title: Set Password Strength Minimum Different Characters ident: CCE-26615-5 severity: low ---

Requiring a minimum number of different characters during password changes ensures that newly changed passwords should not resemble previously compromised ones. Note that passwords which are changed on compromised systems will still be compromised, however.

Failing to set password strength minimum different characters is of low severity.

View all 7 Set Password Quality Requirements, if using pam_cracklib rules.

Technical Description

The pam_cracklib module's difok parameter controls requirements for usage of different characters during a password change. Add difok=NUM after pam_cracklib.so to require differing characters when changing passwords, substituting NUM appropriately. The DoD requirement is 4.
