--- layout: control permalink: /CCE-26435-8/ name: CCE-26435-8 title: Ensure /tmp Located On Separate Partition ident: CCE-26435-8 severity: low ---

The /tmp partition is used as temporary storage by many programs. Placing /tmp in its own partition enables the setting of more restrictive mount options, which can help protect programs which use it.

Failing to ensure /tmp located on separate partition is of low severity.

View all 6 Disk Partitioning rules.

Technical Description

The /tmp directory is a world-writable directory used for temporary file storage. Ensure it has its own partition or logical volume at installation time, or migrate it using LVM.
