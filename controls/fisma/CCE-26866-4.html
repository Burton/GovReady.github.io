--- layout: control permalink: /CCE-26866-4/ name: CCE-26866-4 title: Disable Kernel Parameter for IP Forwarding ident: CCE-26866-4 severity: medium ---

IP forwarding permits the kernel to forward packets from one network interface to another. The ability to forward packets between two networks is only appropriate for systems acting as routers.

Failing to disable kernel parameter for ip forwarding is of medium severity.

View all 3 Network Parameters for Hosts Only rules.

Technical Description

To set the runtime status of the net.ipv4.ip_forward kernel parameter, run the following command: # sysctl -w net.ipv4.ip_forward=0 If this is not the system's default value, add the following line to /etc/sysctl.conf: net.ipv4.ip_forward = 0
