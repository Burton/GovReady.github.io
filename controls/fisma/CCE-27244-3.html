--- layout: control permalink: /CCE-27244-3/ name: CCE-27244-3 title: System Audit Logs Must Be Owned By Root ident: CCE-27244-3 severity: low ---

Failure to give ownership of the audit log files to root allows the designated owner, and unauthorized users, potential access to sensitive information.

Failing to system audit logs must be owned by root is of low severity.

View all 14 Configure auditd Rules for Comprehensive Auditing rules.

Technical Description

To properly set the owner of /var/log, run the command: # chown root /var/log
