--- layout: control permalink: /CCE-26690-8/ name: CCE-26690-8 title: Configure LDAP Client to Use TLS For All Transactions ident: CCE-26690-8 severity: medium ---

The ssl directive specifies whether to use ssl or not. If not specified it will default to no. It should be set to start_tls rather than doing LDAP over SSL.

Failing to configure ldap client to use tls for all transactions is of medium severity.

View all 2 Configure OpenLDAP Clients rules.

Technical Description

Configure LDAP to enforce TLS use. First, edit the file /etc/pam_ldap.conf, and add or correct the following lines: 

ssl start_tls
Then review the LDAP server and ensure TLS has been configured.
