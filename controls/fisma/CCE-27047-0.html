--- layout: control permalink: /CCE-27047-0/ name: CCE-27047-0 title: Restrict Serial Port Root Logins ident: CCE-27047-0 severity: low ---

Preventing direct root login to serial port interfaces helps ensure accountability for actions taken on the systems using the root account.

Failing to restrict serial port root logins is of low severity.

View all 7 Restrict Root Logins rules.

Technical Description

To restrict root logins on serial ports, ensure lines of this form do not appear in /etc/securetty: 

ttyS0
ttyS1
