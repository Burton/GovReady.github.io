--- layout: control permalink: /CCE-27228-6/ name: CCE-27228-6 title: Set Password Hashing Algorithm in /etc/login.defs ident: CCE-27228-6 severity: medium ---

Using a stronger hashing algorithm makes password cracking attacks more difficult.

Failing to set password hashing algorithm in /etc/login.defs is of medium severity.

View all 3 Set Password Hashing Algorithm rules.

Technical Description

In /etc/login.defs, add or correct the following line to ensure the system will use SHA-512 as the hashing algorithm: 

ENCRYPT_METHOD SHA512
