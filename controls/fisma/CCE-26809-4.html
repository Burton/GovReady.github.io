Failing to ensure rsyslog is installed is of medium severity.

Rationale

The rsyslog package provides the rsyslog daemon, which provides system logging services.

via FISMA SSG. View all 3 Configure Syslog rules.

Technical Description

Rsyslog is installed by default. The rsyslog package can be installed with the following command: # yum install rsyslog
