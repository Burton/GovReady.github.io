Failing to uninstall dhcp server package is of medium severity.

Rationale

Removing the DHCP server ensures that it cannot be easily or accidentally reactivated and disrupt network operation.

via FISMA SSG. View all 2 Disable DHCP Server rules.

Technical Description

If the system does not need to act as a DHCP server, the dhcp package can be uninstalled. The dhcp package can be removed with the following command: # yum erase dhcp
