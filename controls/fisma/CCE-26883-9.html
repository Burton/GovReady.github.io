--- layout: control permalink: /CCE-26883-9/ name: CCE-26883-9 title: Enable Kernel Parameter to Ignore ICMP Broadcast Echo Requests ident: CCE-26883-9 severity: low rule_id: sysctl_net_ipv4_icmp_echo_ignore_broadcasts ---

Failing to enable kernel parameter to ignore icmp broadcast echo requests is of low severity.

Rationale

Ignoring ICMP echo requests (pings) sent to broadcast or multicast addresses makes the system slightly more difficult to enumerate on the network.

via FISMA SSG. View all 12 Network Related Kernel Runtime Parameters for Hosts and Routers rules.

Technical Description

To set the runtime status of the net.ipv4.icmp_echo_ignore_broadcasts kernel parameter, run the following command: # sysctl -w net.ipv4.icmp_echo_ignore_broadcasts=1 If this is not the system's default value, add the following line to /etc/sysctl.conf: net.ipv4.icmp_echo_ignore_broadcasts = 1
