--- layout: control permalink: /CCE-27046-2/ name: CCE-27046-2 title: Disable xinetd Service ident: CCE-27046-2 severity: medium ---

The xinetd service provides a dedicated listener service for some programs, which is no longer necessary for commonly-used network services. Disabling it ensures that these uncommon services are not running, and also prevents attacks against xinetd itself.

Failing to disable xinetd service is of medium severity.

View all 2 Xinetd rules.

Technical Description

The xinetd service can be disabled with the following command: # chkconfig xinetd off
