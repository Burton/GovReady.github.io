--- layout: control permalink: /CCE-26948-0/ name: CCE-26948-0 title: Disable vsftpd Service ident: CCE-26948-0 severity: low ---

Running FTP server software provides a network-based avenue of attack, and should be disabled if not needed. Furthermore, the FTP protocol is unencrypted and creates a risk of compromising sensitive information.

Failing to disable vsftpd service is of low severity.

View all 2 Disable vsftpd if Possible rules.

Technical Description

The vsftpd service can be disabled with the following command: # chkconfig vsftpd off
