--- layout: control permalink: /CCE-27030-6/ name: CCE-27030-6 title: Uninstall bind Package ident: CCE-27030-6 severity: low ---

If there is no need to make DNS server software available, removing it provides a safeguard against its activation.

Failing to uninstall bind package is of low severity.

View all 2 Disable DNS Server rules.

Technical Description

To remove the bind package, which contains the named service, run the following command: 

# yum erase bind
