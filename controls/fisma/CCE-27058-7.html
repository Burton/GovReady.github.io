Ensuring the auditd service is active ensures audit records generated by the kernel can be written to disk, or that appropriate actions will be taken if other obstacles exist.

Failing to enable auditd service is of medium severity.

View all 2 System Accounting with auditd rules.

Technical Description

The auditd service is an essential userspace component of the Linux Auditing System, as it is responsible for writing audit records to disk. The auditd service can be enabled with the following command: # chkconfig --level 2345 auditd on
