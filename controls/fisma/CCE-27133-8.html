Failing to uninstall httpd package is of low severity.

Rationale

If there is no need to make the web server software available, removing it provides a safeguard against its activation.

via FISMA SSG. View all 2 Disable Apache if Possible rules.

Technical Description

The httpd package can be removed with the following command: # yum erase httpd
