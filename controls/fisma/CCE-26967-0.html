--- layout: control permalink: /CCE-26967-0/ name: CCE-26967-0 title: Verify Group Who Owns shadow File ident: CCE-26967-0 severity: medium ---

The /etc/shadow file stores password hashes. Protection of this file is critical for system security.

Failing to verify group who owns shadow file is of medium severity.

View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the group owner of /etc/shadow, run the command: # chgrp root /etc/shadow
