The iptables service provides the system's host-based firewalling capability for IPv4 and ICMP.

Failing to verify iptables enabled is of medium severity.

View all 3 Inspect and Activate Default Rules rules.

Technical Description

The iptables service can be enabled with the following command: # chkconfig --level 2345 iptables on
