--- layout: control permalink: /CCE-26954-8/ name: CCE-26954-8 title: Verify Permissions on group File ident: CCE-26954-8 severity: medium ---

The /etc/group file contains information regarding groups that are configured on the system. Protection of this file is important for system security.

Failing to verify permissions on group file is of medium severity.

View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the permissions of /etc/group, run the command: # chmod 644 /etc/group
