--- layout: control permalink: /CCE-27026-4/ name: CCE-27026-4 title: Verify User Who Owns gshadow File ident: CCE-27026-4 severity: medium ---

The /etc/gshadow file contains group password hashes. Protection of this file is critical for system security.

Failing to verify user who owns gshadow file is of medium severity.

View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the owner of /etc/gshadow, run the command: # chown root /etc/gshadow
