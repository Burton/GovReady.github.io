--- layout: control permalink: /CCE-27164-3/ name: CCE-27164-3 title: Disable Accepting IPv6 Router Advertisements ident: CCE-27164-3 severity: low rule_id: sysctl_net_ipv6_conf_default_accept_ra ---

Failing to disable accepting ipv6 router advertisements is of low severity.

Rationale

An illicit router advertisement message could result in a man-in-the-middle attack.

via FISMA SSG. View all 2 Disable Automatic Configuration rules.

Technical Description

To set the runtime status of the net.ipv6.conf.default.accept_ra kernel parameter, run the following command: # sysctl -w net.ipv6.conf.default.accept_ra=0 If this is not the system's default value, add the following line to /etc/sysctl.conf: net.ipv6.conf.default.accept_ra = 0
