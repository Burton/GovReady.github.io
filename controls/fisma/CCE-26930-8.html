Failing to verify group who owns group file is of medium severity.

Rationale

The /etc/group file contains information regarding groups that are configured on the system. Protection of this file is important for system security.

via FISMA SSG. View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the group owner of /etc/group, run the command: # chgrp root /etc/group
