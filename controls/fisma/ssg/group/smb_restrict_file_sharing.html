Restrict SMB File Sharing to Configured Networks Rules

Only users with local user accounts will be able to log in to Samba shares by default. Shares can be limited to particular users or network addresses. Use the and directives accordingly, and consider setting the valid users directive to a limited subset of users or to a group of users.
