Disable Wireless Through Software Configuration Rules

If it is impossible to remove the wireless hardware from the device in question, disable as much of it as possible through software. The following methods can disable software support for wireless networking, but note that these methods do not prevent malicious software or careless users from re-activating the devices.

Count CCE Rule ID Severity Group
1 CCE-26878-9 wireless_disable_in_bios low wireless_software (4)
2 CCE-27057-9 deactivate_wireless_interfaces low wireless_software (4)
3 CCE-27081-9 service_bluetooth_disabled medium wireless_software (4)
4 CCE-26763-3 kernel_module_bluetooth_disabled medium wireless_software (4)
