Use Access Lists to Enforce Authorization Restrictions Rules

When configuring NFS exports, ensure that each export line in contains a list of hosts which are allowed to access that export. If no hosts are specified on an export line, then that export is available to any remote host which requests it. All lines of the exports file should specify the hosts (or subnets, if needed) which are allowed to access the exported directory, so that unknown or remote hosts will be denied.
