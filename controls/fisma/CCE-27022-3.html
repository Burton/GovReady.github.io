--- layout: control permalink: /CCE-27022-3/ name: CCE-27022-3 title: Verify /etc/grub.conf Group Ownership ident: CCE-27022-3 severity: medium ---

The root group is a highly-privileged group. Furthermore, the group-owner of this file should not have any access privileges anyway.

Failing to verify /etc/grub.conf group ownership is of medium severity.

View all 4 Set Boot Loader Password rules.

Technical Description

The file /etc/grub.conf should be group-owned by the root group to prevent destruction or modification of the file. To properly set the group owner of /etc/grub.conf, run the command: # chgrp root /etc/grub.conf
