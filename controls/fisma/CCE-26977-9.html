Failing to uninstall squid package is of low severity.

Rationale

If there is no need to make the proxy server software available, removing it provides a safeguard against its activation.

via FISMA SSG. View all 2 Disable Squid if Possible rules.

Technical Description

The squid package can be removed with the following command: # yum erase squid
