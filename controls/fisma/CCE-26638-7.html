Failing to implement blank screen saver is of low severity.

Rationale

Setting the screensaver mode to blank-only conceals the contents of the display from passersby.

via FISMA SSG. View all 4 Configure GUI Screen Locking rules.

Technical Description

Run the following command to set the screensaver mode in the GNOME desktop to a blank screen: 

# gconftool-2 --direct \
  --config-source xml:readwrite:/etc/gconf/gconf.xml.mandatory \
  --type string \
  --set /apps/gnome-screensaver/mode blank-only
