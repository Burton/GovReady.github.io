--- layout: control permalink: /CCE-26975-3/ name: CCE-26975-3 title: Verify Group Who Owns gshadow File ident: CCE-26975-3 severity: medium rule_id: groupowner_gshadow_file ---

Failing to verify group who owns gshadow file is of medium severity.

Rationale

The /etc/gshadow file contains group password hashes. Protection of this file is critical for system security.

via FISMA SSG. View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the group owner of /etc/gshadow, run the command: # chgrp root /etc/gshadow
