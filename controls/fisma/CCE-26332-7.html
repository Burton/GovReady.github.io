Failing to uninstall net-snmp package is of low severity.

Rationale

If there is no need to run SNMP server software, removing the package provides a safeguard against its activation.

via FISMA SSG. View all 2 Disable SNMP Server if Possible rules.

Technical Description

The net-snmp package provides the snmpd service. The net-snmpd package can be removed with the following command: # yum erase net-snmpd
