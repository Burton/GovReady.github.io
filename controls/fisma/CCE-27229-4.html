--- layout: control permalink: /CCE-27229-4/ name: CCE-27229-4 title: Set Password Hashing Algorithm in /etc/libuser.conf ident: CCE-27229-4 severity: medium ---

Using a stronger hashing algorithm makes password cracking attacks more difficult.

Failing to set password hashing algorithm in /etc/libuser.conf is of medium severity.

View all 3 Set Password Hashing Algorithm rules.

Technical Description

In /etc/libuser.conf, add or correct the following line in its [defaults] section to ensure the system will use the SHA-512 algorithm for password hashing: 

crypt_style = sha512
