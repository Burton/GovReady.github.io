--- layout: control permalink: /CCE-26868-0/ name: CCE-26868-0 title: Verify Permissions on passwd File ident: CCE-26868-0 severity: medium ---

If the /etc/passwd file is writable by a group-owner or the world the risk of its compromise is increased. The file contains the list of accounts on the system and associated information, and protection of this file is critical for system security.

Failing to verify permissions on passwd file is of medium severity.

View all 12 Verify Permissions on Files with Local Account Information and Credentials rules.

Technical Description

To properly set the permissions of /etc/passwd, run the command: # chmod 0644 /etc/passwd
