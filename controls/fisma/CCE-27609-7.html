--- layout: control permalink: /CCE-27609-7/ name: CCE-27609-7 title: Ensure All Accounts on the System Have Unique Names ident: CCE-27609-7 severity: low ---

Unique usernames allow for accountability on the system.

Failing to ensure all accounts on the system have unique names is of low severity.

View all 3 Set Account Expiration Parameters rules.

Technical Description

Change usernames, or delete accounts, so each has a unique name.
