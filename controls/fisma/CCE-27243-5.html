--- layout: control permalink: /CCE-27243-5/ name: CCE-27243-5 title: System Audit Logs Must Have Mode 0640 or Less Permissive ident: CCE-27243-5 severity: low ---

If users can write to audit logs, audit trails can be modified or destroyed.

Failing to system audit logs must have mode 0640 or less permissive is of low severity.

View all 14 Configure auditd Rules for Comprehensive Auditing rules.

Technical Description

Change the mode of the audit log files with the following command: 

# chmod 0640 audit_file
