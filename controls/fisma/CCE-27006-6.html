--- layout: control permalink: /CCE-27006-6/ name: CCE-27006-6 title: Verify ip6tables Enabled if Using IPv6 ident: CCE-27006-6 severity: medium ---

The ip6tables service provides the system's host-based firewalling capability for IPv6 and ICMPv6.

Failing to verify ip6tables enabled if using ipv6 is of medium severity.

View all 3 Inspect and Activate Default Rules rules.

Technical Description

The ip6tables service can be enabled with the following command: # chkconfig --level 2345 ip6tables on
