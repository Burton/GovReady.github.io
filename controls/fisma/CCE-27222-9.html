--- layout: control permalink: /CCE-27222-9/ name: CCE-27222-9 title: Configure Periodic Execution of AIDE ident: CCE-27222-9 severity: medium ---

By default, AIDE does not install itself for periodic execution. Periodically running AIDE is necessary to reveal unexpected changes in installed files.

Failing to configure periodic execution of aide is of medium severity.

View all 4 Verify Integrity with AIDE rules.

Technical Description

To implement a daily execution of AIDE at 4:05am using cron, add the following line to /etc/crontab: 

05 4 * * * root /usr/sbin/aide --check
AIDE can be executed periodically through other means; this is merely one example.
