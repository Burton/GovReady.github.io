Failing to disable dovecot service is of low severity.

Rationale

Running an IMAP or POP3 server provides a network-based avenue of attack, and should be disabled if not needed.

via FISMA SSG. View all 2 Disable Dovecot rules.

Technical Description

The dovecot service can be disabled with the following command: # chkconfig dovecot off
