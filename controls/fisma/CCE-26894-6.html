Failing to disable ypbind service is of medium severity.

Rationale

Disabling the ypbind service ensures the system is not acting as a client in a NIS or NIS+ domain.

via FISMA SSG. View all 2 NIS rules.

Technical Description

The ypbind service, which allows the system to act as a client in a NIS or NIS+ domain, should be disabled. The ypbind service can be disabled with the following command: # chkconfig ypbind off
