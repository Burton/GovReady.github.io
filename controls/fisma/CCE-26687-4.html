Failing to uninstall vsftpd package is of low severity.

Rationale

Removing the vsftpd package decreases the risk of its accidental activation.

via FISMA SSG. View all 2 Disable vsftpd if Possible rules.

Technical Description

The vsftpd package can be removed with the following command: # yum erase vsftpd
