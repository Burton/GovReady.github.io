Failing to disable mounting of cramfs is of low severity.

Rationale

Linux kernel modules which implement filesystems that are not needed by the local system should be disabled.

via FISMA SSG. View all 14 Restrict Dynamic Mounting and Unmounting of Filesystems rules.

Technical Description

To configure the system to prevent the cramfs kernel module from being loaded, add the following line to a file in the directory /etc/modprobe.d: install cramfs /bin/false This effectively prevents usage of this uncommon filesystem.
