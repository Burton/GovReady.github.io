Failing to disable samba is of low severity.

Rationale

Running a Samba server provides a network-based avenue of attack, and should be disabled if not needed.

via FISMA SSG. View all 1 Disable Samba if Possible rules.

Technical Description

The smb service can be disabled with the following command: # chkconfig smb off
