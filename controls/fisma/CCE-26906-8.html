--- layout: control permalink: /CCE-26906-8/ name: CCE-26906-8 title: Disable snmpd Service ident: CCE-26906-8 severity: low ---

Running SNMP software provides a network-based avenue of attack, and should be disabled if not needed.

Failing to disable snmpd service is of low severity.

View all 2 Disable SNMP Server if Possible rules.

Technical Description

The snmpd service can be disabled with the following command: # chkconfig snmpd off
