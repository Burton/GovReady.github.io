Failing to disable interface usage of ipv6 is of low severity.

Rationale

via FISMA SSG. View all 3 Disable Support for IPv6 Unless Needed rules.

Technical Description

To disable interface usage of IPv6, add or correct the following lines in /etc/sysconfig/network: 

NETWORKING_IPV6=no
IPV6INIT=no
