--- layout: control permalink: /CCE-27265-8/ name: CCE-27265-8 title: Disable System Statistics Reset Service (sysstat) ident: CCE-27265-8 severity: low ---

By default the sysstat service merely runs a program at boot to reset the statistics, which can be retrieved using programs such as sar and sadc. These may provide useful insight into system operation, but unless used this service can be disabled.

Failing to disable system statistics reset service (sysstat) is of low severity.

View all 24 Base Services rules.

Technical Description

The sysstat service resets various I/O and CPU performance statistics to zero in order to begin counting from a fresh state at boot time. The sysstat service can be disabled with the following command: # chkconfig sysstat off
