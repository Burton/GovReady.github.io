Failing to disable httpd service is of low severity.

Rationale

Running web server software provides a network-based avenue of attack, and should be disabled if not needed.

via FISMA SSG. View all 2 Disable Apache if Possible rules.

Technical Description

The httpd service can be disabled with the following command: # chkconfig httpd off
