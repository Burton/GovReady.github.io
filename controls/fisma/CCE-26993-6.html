--- layout: control permalink: /CCE-26993-6/ name: CCE-26993-6 title: Enable Kernel Parameter to Ignore Bogus ICMP Error Responses ident: CCE-26993-6 severity: low rule_id: sysctl_net_ipv4_icmp_ignore_bogus_error_responses ---

Failing to enable kernel parameter to ignore bogus icmp error responses is of low severity.

Rationale

Ignoring bogus ICMP error responses reduces log size, although some activity would not be logged.

via FISMA SSG. View all 12 Network Related Kernel Runtime Parameters for Hosts and Routers rules.

Technical Description

To set the runtime status of the net.ipv4.icmp_ignore_bogus_error_responses kernel parameter, run the following command: # sysctl -w net.ipv4.icmp_ignore_bogus_error_responses=1 If this is not the system's default value, add the following line to /etc/sysctl.conf: net.ipv4.icmp_ignore_bogus_error_responses = 1
