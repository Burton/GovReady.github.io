--- layout: control permalink: /CCE-27073-6/ name: CCE-27073-6 title: Uninstall telnet-server Package ident: CCE-27073-6 severity: high ---

Removing the telnet-server package decreases the risk of the telnet service's accidental (or intentional) activation.

Failing to uninstall telnet-server package is of high severity.

View all 2 Telnet rules.

Technical Description

The telnet-server package can be uninstalled with the following command: 

# yum erase telnet-server
