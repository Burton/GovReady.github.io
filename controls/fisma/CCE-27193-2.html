--- layout: control permalink: /CCE-27193-2/ name: CCE-27193-2 title: Disable Software RAID Monitor (mdmonitor) ident: CCE-27193-2 severity: low ---

If software RAID monitoring is not required, there is no need to run this service.

Failing to disable software raid monitor (mdmonitor) is of low severity.

View all 24 Base Services rules.

Technical Description

The mdmonitor service is used for monitoring a software RAID array; hardware RAID setups do not use this service. The mdmonitor service can be disabled with the following command: # chkconfig mdmonitor off
