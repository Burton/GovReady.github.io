--- layout: control permalink: /CCE-27005-8/ name: CCE-27005-8 title: Uninstall xinetd Package ident: CCE-27005-8 severity: low ---

Removing the xinetd package decreases the risk of the xinetd service's accidental (or intentional) activation.

Failing to uninstall xinetd package is of low severity.

View all 2 Xinetd rules.

Technical Description

The xinetd package can be uninstalled with the following command: 

# yum erase xinetd
