--- layout: control permalink: /CCE-27039-7/ name: CCE-27039-7 title: Uninstall dovecot Package ident: CCE-27039-7 severity: low ---

If there is no need to make the Dovecot software available, removing it provides a safeguard against its activation.

Failing to uninstall dovecot package is of low severity.

View all 2 Disable Dovecot rules.

Technical Description

The dovecot package can be uninstalled with the following command: 

# yum erase dovecot
