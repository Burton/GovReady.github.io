--- layout: control permalink: /CCE-27150-2/ name: CCE-27150-2 title: Set Permissions on the /var/log/httpd/ Directory ident: CCE-27150-2 severity: low ---

Access to the web server's log files may allow an unauthorized user or attacker to access information about the web server or alter the server's log files.

Failing to set permissions on the /var/log/httpd/ directory is of low severity.

View all 3 Restrict File and Directory Access rules.

Technical Description

Ensure that the permissions on the web server log directory is set to 700: 

# chmod 700 /var/log/httpd/
This is its default setting.
