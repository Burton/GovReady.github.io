Failing to allow only ssh protocol 2 is of high severity.

Rationale

SSH protocol version 1 suffers from design flaws that result in security vulnerabilities and should not be used.

via FISMA SSG. View all 11 Configure OpenSSH Server if Necessary rules.

Technical Description

Only SSH protocol version 2 connections should be permitted. The default setting in /etc/ssh/sshd_config is correct, and can be verified by ensuring that the following line appears: 

Protocol 2
