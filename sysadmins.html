GovReady for Sys Admins & Security

GovReady is a pure bash shell script wrapper around OpenScap and the Fedora SCAP-Security-Guide.

GovReady is currently pre-release and currently requires root privileges to install and run. (We are working on that.) The source code is simple enough for manual review. We invite you to try GovReady and provide feedback.

We invite you to try GovReady and provide feedback.

Get started

Install on a vagrant box

vagrant ssh

su -

curl -Lk https://raw.githubusercontent.com/GovReady/govready/master/install.sh | sudo bash

govready install_ssg

govready scan

ls usgcb-rhel6-server-results-*.html

lynx usgcb-rhel6-server-results-*.html

Install on your server

ssh you@yourdevserver

su -

curl -Lk https://raw.githubusercontent.com/GovReady/govready/master/install.sh | sudo bash

govready install_ssg

govready scan

ls usgcb-rhel6-server-results-*.html

lynx usgcb-rhel6-server-results-*.html
